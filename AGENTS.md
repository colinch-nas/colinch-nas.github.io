# Colinch NAS 경량 하네스

- 공통 규칙은 상위 `AGENTS.md`를 따른다. 자동 포스팅 중 `CLAUDE.md` 전체를 읽지 않는다.
- 언어: 한국어 직접체 `~다`. 실제 장비·버전·삽질 포인트가 있는 설정 글처럼 쓴다.
- 주제: NAS 설정, 홈서버·홈랩, 자체 호스팅, Home Assistant, 네트워크·보안.
- 웹 검색: 최신 제품, 펌웨어, Home Assistant/Proxmox/TrueNAS 같은 버전 이슈가 핵심이면 1~2회 확인한다.
- SEO: 제품명·버전·작업 결과를 title·description·첫 문단에 넣는다.
- 태그 후보: `Synology`, `QNAP`, `TrueNAS`, `TerraMaster`, `DSM`, `Proxmox`, `Docker`, `Ubuntu`, `HomeLab`, `HomeAssistant`, `Zigbee`, `Matter`, `Jellyfin`, `Nextcloud`, `Vaultwarden`, `Immich`, `Tailscale`, `CloudflareTunnel`, `NAS보안`.
- 설정 글은 환경 정보(장비, OS/버전, 네트워크 조건)를 짧게 적고, 명령어 위에는 목적을 한 줄로 쓴다.
- 시각 자료: 네트워크 구성도, 설정 단계 표, 장비·OS 비교표, 포트/권한 체크리스트를 우선한다.
- 보안 글은 외부 공개, 포트포워딩, 관리자 계정, 백업 리스크를 빠뜨리지 않는다.
