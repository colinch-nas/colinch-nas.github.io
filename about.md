**NAS·홈서버·스마트홈 실전 활용 블로그입니다.**

----------

NAS를 사면서 처음에는 그냥 파일 저장 용도로만 쓰다가, 어느 순간부터 Docker 올리고 미디어 서버 만들고 Home Assistant까지 돌리게 됐다. 설정할 때마다 한국어 자료가 부족하거나 오래된 버전 기준이어서 삽질했던 경험을 정리해둔다.

**1. NAS 설정 가이드 — Synology·QNAP·TrueNAS**

처음 구매한 사람도 따라할 수 있도록, 실제 설정 화면 경로와 명령어 중심으로 쓴다. 버전이 바뀌면 달라지는 부분도 함께 표기한다.

**2. 홈서버·홈랩 구축**

미니 PC나 구형 노트북으로 Proxmox, Docker 서버를 만드는 법. 소비전력과 성능 사이에서 고민했던 것들도 포함한다.

**3. 자체 호스팅 서비스**

Jellyfin, Nextcloud, Vaultwarden, Immich 등 클라우드 서비스를 대체하거나 보완하는 서비스를 직접 운영하는 방법이다. 설치보다 운영 중 생기는 문제 해결이 더 어렵다.

**4. Home Assistant 스마트홈**

스마트 전구 하나 연결하려고 시작했다가 집 전체 자동화가 됐다. Zigbee2MQTT, Matter, 에너지 모니터링까지 실제로 쓰는 설정만 다룬다.

**5. 보안·네트워크**

Tailscale VPN, Cloudflare Tunnel, Nginx Proxy Manager 등 외부에서 안전하게 접속하는 방법. NAS 보안 설정 실수로 랜섬웨어 피해를 입는 사례가 늘고 있어 이 부분은 빠뜨리지 않는다.

----------

잘못된 설정이나 최신 버전에서 달라진 부분이 있으면 댓글로 알려주면 반영한다.
